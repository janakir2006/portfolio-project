Mini Project Report

Project Title

Simple Portfolio Website

Objective

To design and develop a simple, responsive personal portfolio website using HTML and CSS that showcases personal information, projects, resume, and contact details.

Tools & Technologies Used

  • HTML5 for webpage structure
  • CSS3 for styling and layout
  • Google Fonts for aesthetic typography
  • Boxicons for social media icons
  • VS Code code editor
  • GitHub Pages for deployment (optional)

Features

  • Fixed and responsive navigation bar
  • Profile image and personal intro section
  • Smooth scrolling between sections (About, Projects, Contact)
  • Projects section with clickable previews
  • Contact form with input fields
  • Social media integration (LinkedIn, GitHub)
  • Responsive design for desktop and mobile views

Project Structure

Sections Included:

  • Header with name, navigation links, and social icons
  • Intro Section with profile image and personal intro
  • Projects a visual showcase of 3 main projects
  • Contact Form to collect user messages
  • Footer with additional links and copyright

Code Overview

HTML

Structured using div, section, and header tags. Semantic elements used for clarity and SEO.

CSS

Custom styles for layout, colors, fonts, spacing, and media queries for responsiveness.

Challenges Faced

  • Making the design responsive across different screen sizes
  • Integrating custom fonts and icons properly
  • Adjusting layout for mobile view without breaking styles

Future Enhancements

  • Add dark mode
  • Store contact form data using JavaScript or backend
  • Include skill bars and testimonials section
  • Deploy the site on GitHub Pages or Netlify

Conclusion

This project helped apply my knowledge of HTML and CSS practically. I learned how to build a clean, responsive, and personal web portfolio that represents my identity online.
